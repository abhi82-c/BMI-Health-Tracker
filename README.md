# Student Health & BMI Tracker

### Project Description
This is a web-based application developed as a minor project for Apex University. It allows users to enter their height and weight to calculate their Body Mass Index (BMI). The application displays the BMI value along with a basic health category such as Underweight, Healthy, Overweight, or Obese.

### Technologies Used
- HTML5
- CSS3
- JavaScript
